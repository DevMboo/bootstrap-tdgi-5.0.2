Explicando a montagem do HTML

Dividindo por etapas
  • As tag de coloração do bootstrap adaptado TDGI seguem o formato padrão do próprio bootstrap, tendo como única mudança o final da class, sendo assim então uma class que é inserida como bg-dark ou bg-primary ao terem o final inserido -td, passam a ter a estilização elaborada pelo a adaptação do framework CSS, e esse padrão se mantem por todas as ferramentas que foram adaptadas. As adaptações são nos seguintes padrões bg-primary-td, bg-danger-td, bg-warning-td e bg-darklight-td , no exemplo abaixo utilizei a tag < nav>, entretando qualquer controlador de fluxo pode ser utilizado com esses padrões de class.
    Exemplo:

    Código pronto:


    < nav class="col-md-12 bg-darklight-td">
    < p class="text-white">bg-darklight-td< /p> < /nav>

    Saída:

    bg-darklight-td
