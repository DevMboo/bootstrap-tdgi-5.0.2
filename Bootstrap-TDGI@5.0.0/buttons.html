Explicando a montagem do HTML

Dividindo em etapas
  • Os botões devem sempre ser compostos com 4 tags de controle de fluxo dentro de uma tag < a class="btn-success-td">texto qualquer< /a>, essas tags podem ser < span> ou < div> sendo que esses container's devem ser acompanhados da class="line", e seguindo a regra de enumeração do 1 até o 4, que formaram as 4 bordas do botão, sendo assim elas devem ser numeradas da seguinte forma: < span class='line-1-success'>< /span>, < span class='line-2-success'>< /span>, < span class='line-3-success'>< /span>, < span class='line-4-success'>< /span>, essas tags span ou div serão manipuladas pelo CSS e vão rotacionar assim que a janela for carregada para o usuário. E para que a animação e coloração funcione corretamente precisa ser definido a designação da estilização desejada, sendo elas "primary","warning","success" e "danger".
    Lembre sempre!, todas as TAGS precisam ser envolvidas em uma < div> com class="btn-cont", ela sim vai envolver todos os componentes que animam a ferramenta.
    Exemplo:

    Código pronto:


    < div class='btn-cont'>
    < a class='btn-success-td btn' href='#'>
     btn-success-td
    < span class='line-1-success'>< /span>
    < span class='line-2-success'>< /span>
    < span class='line-3-success'>< /span>
    < span class='line-4-success'>< /span>
    < /a>
    < /div>

    Saída:

    btn-success-td
